# 圖片檔名

把圖片放在這個資料夾，遊戲會優先使用這些檔案：

- `chinchinko-title.png`：主選單整張開場封面
- `title-cover.png`：主選單封面備用圖
- `character-select.png`：角色三選一整張圖片
- `chinchinko-profile.png`：點選親親子後顯示的人物介紹圖
- `title-left-locked.png`：左側未解鎖角色剪影
- `title-right-locked.png`：右側未解鎖角色剪影
- `character-select.png`：角色三選一合圖，備用
- `xiao-zhengming.png`：蕭證明休息室整張背景圖
- `ugly-pillow.png`：醜臉抱枕圖

如果圖片不存在，遊戲會使用目前的 CSS 佔位圖。
